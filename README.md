## 画像・資料アーカイブ（MECE 分類）

学習・業務・思考整理で使う画像／PDF のアーカイブです。すべて日本語のカテゴリで整理されています。
Cursor×gpt-5 で実装しました。

### 目的

- 必要な図解・資料へ最短でアクセスするための共通棚を用意
- 追加や改編に強い「重複なくモレの少ない」分類体系（MECE）を採用
